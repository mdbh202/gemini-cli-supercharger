# Contributing to Gemini CLI Supercharger

Thank you for your interest in making Gemini CLI even faster! We welcome contributions from the community.

## How to Contribute

1.  **Report Bugs:** If you find a case where the Supercharger breaks a hook or UI element, please open an Issue.
2.  **Suggest Optimizations:** Have a better way to bypass the shell or optimize the V8 cache? Let us know!
3.  **Submit Pull Requests:** 
    *   Fork the repo.
    *   Create a feature branch.
    *   Ensure your changes are tested on macOS.
    *   Submit your PR with a clear description of the performance gain.

## Development Setup

To test your changes locally:
1.  Run `./scripts/apply-patch.sh` to apply your modifications.
2.  Use the included `./scripts/benchmark.sh` to verify the speed improvement.

## Code of Conduct

Be professional and respectful to all contributors. Our goal is to build a high-performance tool for everyone.
